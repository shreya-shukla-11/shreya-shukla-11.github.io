Agents on Steroids

This one is my favorite project. This is a text-to-SQL bot but it did a lot more than that. We built our in-house visualization library that, given a CSV, transforms it into 2-3 most suitable graphs for that kind of data. It also had an RCA library that did RCA for north star metrics and a RAG system for a new employee to learn logs (it was text-to-SQL for the logs team).
